== Description ==

A plugin by Spiraclethemes.com to extend its free themes with one-click demo data import functionality.

How to use this Spiraclethemes site library?

1. While installing free themes by Spiraclethemes, kindly install & activate our plugin "Spiraclethemes Site Library" or do this manually.
2. To import demo content, go to Appearance > Import Demo Data.
3. Preview and select your desired demo, then click the import button.
4. Contact Spiraclethemes support at http://support.spiraclethemes.com/ for assistance.

== Installation ==

*** From your WordPress dashboard ***

1. Visit 'Plugins > Add New',
2. Search for 'Spiraclethemes Site Library' and install the plugin.
3. Activate 'Spiraclethemes Site Library' from your Plugins page.

== Frequently Asked Questions ==

= Does this plugin work with any WordPress theme? =

No, this plugin works only with themes developed by [Spiraclethemes](https://spiraclethemes.com/).
